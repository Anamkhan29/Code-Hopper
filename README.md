# Code-Hopper: Learn 6 Programming Languages While Playing!

## Overview

Code-Hopper is a fun, interactive Flappy Bird-style game that helps users learn programming concepts across 6 different languages:  

- Java  
- Python  
- JavaScript  
- C++  
- C#  
- Ruby  

Players control a bird using the spacebar to navigate through pipes. Every few points, the game pauses and presents a programming quiz question. Correct answers let the player continue, while wrong answers end the game. A Hint button is available for guidance.

---

## Features

- Classic Flappy Bird gameplay with moving pipes and scoring  
- Quiz questions for 6 programming languages  
- Hint button for each question  
- Score tracking and interactive feedback  
- Browser-based, fully deployable via Google Antigravity  

---

## How to Play

1. Press Spacebar to make the bird flap and avoid pipes.  
2. Every ~10 seconds, a quiz question will appear:
   - Read the question carefully  
   - Click on your answer  
   - Optionally click Hint  to get a clue  
3. Correct answers → continue playing  
4. Wrong answers → game over  
5. Score increases as you pass pipes.
